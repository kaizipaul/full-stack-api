# 📖 Full-Stack-Api<a name="about-project"></a>

> Describe your project in 1 or 2 sentences.

** Full-Stack-Api** is a project consisting of both front end and backend.backend used to create Api data and connect to the database and the front end to grap the api data from the backend abd displays the Api data.Data passed in by the user is being saved in the database

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:


Example command:

```
 bundle install
```


### Setup

Clone this repository to your desired folder:

Example command:

```
  git clone https://github.com/kaizipaul/full-stack-api
  
```

### Install

Install this project with:

```
  bundle install
```

### Usage

To build the project, execute the following command:


Example command:

```
  rails s
```

### Deployment

You can deploy this project using:


Example:

```
Render Platform
Railways
Heroku (Paid)

```
